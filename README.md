

# Pegboard Map

<small>※本ファイル作成にあたり、一部生成AI(VS Code Copilot)を利用しております</small>  
  
有孔板をモチーフにした地図作成ツールです。ピンを配置し、線でつなぐことで、立地や道筋をメモしておけます。ReactとTypeScriptで構築されたWebアプリです。

## 機能

- **ピンの配置とカスタマイズ**: アイコン付きのピンをボード上に配置。名前を設定可能。
- **線接続**: ピン同士を直線、点線、二重線などで接続。
- **マス選択・スクロール**　ボード上をクリックしてマスを選択、ドラッグしてスクロール。
- **履歴管理**: Undo/Redo機能で操作を戻せます。
- **保存/読み込み**: JSON形式でデータを保存・読み込み。localStorage対応。
- **プリセットデータ**: サンプル地図（自宅、駅など）が初期表示されます。
- **幅優先探索**: 「自宅」ピンから到達不可能なピンは薄く表示されます。

## 技術スタック

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: SimpleBar (スクロール), Canvas API (線描画)
- **Icons**: SVG (vite-plugin-svgr使用)

## 使い方

### 1. ピンを設置する
- 画面右側の「ピン」タブを開き、アイコンを選択します。
- ボード上の空きマスをクリックすると、選択したアイコンのピンが設置されます。
- ピンに名前をつけたい場合は、設置後にピンをクリックして「名前を設定」を選択。

### 2. 線を接続する
- 画面右側の「線」タブを開き、線の種類（直線、点線など）を選択します。
- 接続したいピンを順番にクリック：まず始点、次に終点をクリックすると線が引かれます。
- 接続を解除したい場合は、ピンをクリックして「線を外す」を選択。

### 3. ピンを編集・移動する
- 設置済みのピンをクリックするとメニューが表示されます。
- 「ピンを移動」：移動先の空きマスをクリック。
- 「アイコンを変更」：新しいアイコンを選択。
- 「名前を変更」：ピンの名前を編集。
- 注意：自宅ピン（青地、白アイコンのピン）は削除できません。

### 4. ボードを操作する
- **ズーム**: 右下のズームボタンで拡大/縮小（50%～150%）。
- **スクロール**: ボードをドラッグして移動。
- **Undo/Redo**: ヘッダーの矢印アイコンで操作を戻せます。

### 5. 保存・読み込み
- **保存**: ヘッダーのフロッピーディスクアイコンをクリックし、ローカルストレージに保存。
- **JSON入出力**: ヘッダーメニューからJSONの保存・読み込みができます。

### Tips
- 自宅ピンから線でつながっていないピンは薄く表示されます。接続を忘れていないか確認しましょう。
- プリセットデータが初期表示されるので、まずはそれを編集してみてください。
- モバイルではタッチ操作に対応していますが、PCでの操作がおすすめです。
